インストールとアンインストールについて


【インストールの前に．．．】
　StructureT-Makerを実行するためにはMicrosoft .NET Framework 1.1とMicrosoft Excelが
　インストールされている必要があります。
　
　・Microsoft .NET Framework 1.1をインストールする方法
　 OSにより若干違いがありますが、基本的には同じです。
　 １．インターネットエクスプローラを起動し、「ツール」→「Windows Update」をクリック
　 ２．「更新をスキャンする」をクリック
　 ３．画面左にある「Windows」（使用しているOSの名前）をクリック
　 ４．Microsoft .NET Framework 1.1を選択し、「更新の確認とインストール」をクリック
　 ５．「今すぐインストール」をクリック
　 ６．使用許諾に同意する
　 ７．再起動する
　 これでインストールは完了です。
　 セキュリティに関する更新が出ている可能性があるので、念のためもう一度アップデートを
　 行ったほうが良いと思います。



【インストール】
　お好きな場所にダウンロードした圧縮データを解凍して下さい。
　レジストリは一切変更しません。


【アンインストール】
　【インストール】で作ったディレクトリを削除して下さい。
　「送る」に登録している場合は、お手数ですが手動で削除して下さい。
